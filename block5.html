Exercise - Part 1

Create an OpenLayers map with an OpenStreetMap layer. Center the map on the user's location. Add a search box where the user can enter a search term, and the map centers on the location of the first OpenStreetMap Nominatim search result.

Exercise - Part 2

Draw a marker at the most recent user or search result location. Add a button to re-center the map on the user's location. Instead of just centering the map on the new location, fit the map's extent to the accuracy circle of the Geolocation and the bounding box of the search result.
